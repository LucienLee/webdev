`&&` 和 `||`

  • data truthy → 回傳 data.result
  • 保護 data.result
  • 要不然 data 是 undefined 時會有 TypeError: Cannot read property 'result' of undefined

&& 和 || 其實不一定要傳回 true 或 false。

&& : Guard Operator (Logical AND)

`

data && data.result

`

|| : Default Operator (Logical OR)
